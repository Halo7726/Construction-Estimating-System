# Construction-Estimating-System
This repository contains the full source for the S.H. Creel operations portal, a Vercel-based web application with SharePoint as its backend data layer. The system is designed to manage the complete estimating and project lifecycle — from initial opportunity intake through active estimating, bid submission, award, and handoff to project management — with all project records, files, and tasks living in SharePoint and surfaced through a unified Vercel frontend via Microsoft Graph API.
Key integrations include B2W Estimating for live estimate data, contacts, companies, and bid items via API; Office 365 for calendar event management and email distribution; and a planned ZZTakeoff integration for takeoff project setup. An AI-assisted onboarding flow extracts project data from uploaded bid documents with confidence scoring, pre-populating the project intake form for estimator review before any records are created or actions fired.
The estimating module handles opportunity tracking, ITB and RFQ distribution to subcontractors sourced from B2W contacts, proposal collection and tracking, and bid result logging. Status changes in the application trigger automatic SharePoint folder moves, task creation, and Teams notifications. Project records are never deleted — the full history of every opportunity, bid, win, and loss is retained permanently in SharePoint.
Future integrations planned and accounted for in the current data architecture include ZZTakeoff takeoff project automation and Foundation accounting job creation for awarded projects.
